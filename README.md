# for-shannicy

A little website made with love.

A 9-page interactive journey built with React and Three.js: a 3D beating heart, falling rose petals, gold-dust particles, tilt-parallax photos, a deck of "reasons" cards, a heart you press and hold, a sealed letter that opens, a living counter, ambient music, and a finale where every tap is a kiss.

## Run it locally

Just open `index.html` in a browser — no build tools or install needed. React, Three.js, and the fonts load from a CDN, so you need an internet connection.

## Deploy to Vercel (recommended — gives you a link to send her)

**Easiest way (no GitHub needed):**

1. Install the Vercel CLI once: `npm i -g vercel`
2. In this folder, run: `vercel`
3. Log in when prompted, accept the defaults (it's a static site — no build command, no output directory).
4. Run `vercel --prod` to get the final link, something like `https://for-shannicy.vercel.app` — that's the link you send her.

**Or via GitHub:**

1. Push this folder to a GitHub repository.
2. Go to [vercel.com](https://vercel.com) → **Add New → Project** → import the repo.
3. Framework preset: **Other**. Leave build command and output directory empty.
4. Click **Deploy**. Your link appears in under a minute.

Tip: in the Vercel project settings you can rename the deployment so the URL reads nicely — e.g. `for-shannicy.vercel.app`.
